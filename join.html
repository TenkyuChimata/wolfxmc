加入步骤 / How to Join

  1. 打开 Minecraft Java 版 → 多人游戏。 Open Minecraft Java → Multiplayer.
  2. 点击 添加服务器。 Click Add Server.
  3. 服务器地址 填写：wolfx.jp。 Server Address: wolfx.jp.
  4. 保存 → 连接。 Save → Join.

（小提示）进服前建议先看规则页，体验会更顺畅。 (Tip) Read rules first for a smoother start.

常见问题 / Troubleshooting

Q1：提示无法解析地址 / Unknown host？
可能是 DNS 问题。你可以尝试更换 DNS（如 1.1.1.1 / 8.8.8.8），或稍后重试。
Might be a DNS issue. Try another DNS (1.1.1.1 / 8.8.8.8) or retry later.

Q2：版本不匹配 / Incompatible version？
请切换到支持范围内的版本（1.9 - 1.21.11）。推荐使用较新的稳定版本。
Switch to a supported version (1.9 - 1.21.11). Newer stable versions are recommended.
